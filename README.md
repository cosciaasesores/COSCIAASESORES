# Coscia Asesores - Productores de Seguros

Sitio web oficial de Coscia Asesores. 

## Tecnologías
- **Next.js 16** (App Router)
- **Tailwind CSS 4**
- **Framer Motion** (Animaciones)
- **Lucide React** (Iconografía)

## Desarrollo Local

1. Instalar dependencias:
   ```bash
   npm install
   ```

2. Correr el servidor de desarrollo:
   ```bash
   npm run dev
   ```

3. Construir para producción:
   ```bash
   npm run build
   ```

## Estructura del Proyecto
El proyecto sigue una arquitectura de componentes atómicos y estados de resiliencia para asegurar una experiencia premium y robusta.
